# 🐾 宠物MBTI测试

> 🐱🐶 30道题，发现你家主子的隐藏性格！

一个有趣的宠物性格测试，通过观察你家毛孩子的日常行为，测出它的"人格类型"。

## 🎮 怎么玩

1. **开始测试** - 回答30道关于宠物行为的选择题
2. **查看结果** - 获得你家主子的专属人格类型
3. **探索图鉴** - 浏览25种不同的宠物性格类型
4. **分享晒宠** - 把结果分享到朋友圈，看看谁家的毛孩子最特别

## ✨ 特色功能

- 🎯 **25种人格类型** - 从"含羞草"到"六边形战士"，总有一款适合TA
- 📊 **15维性格画像** - 雷达图直观展示宠物的性格特征
- 🎨 **稀有度评级** - 看看你家主子的性格有多特别（传说/史诗/稀有/常见）
- 🌙 **暗色模式** - 护眼模式，晚上也能愉快测试
- 📱 **海报分享** - 一键生成精美结果海报
